# HAZESOFT-T3 - Backup Script

## Description
This repository contains a Bash script `backup.sh` that takes a directory path as an argument and creates a compressed tar archive with a timestamp. The backup is saved in `$HOME/backup`.

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/HAZESOFT-T3.git
